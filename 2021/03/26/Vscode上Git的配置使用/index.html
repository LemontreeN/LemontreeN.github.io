Vscode上Git的配置和使用

Created2021-03-26|Updated2021-05-29
|Post View:

需要的工具：Vscode 和 Git

配置

全部安装完成后，先进入命令行cmd输入git查看安装情况

打开Vscode 点击左侧这样的一个图标

此时右侧有两个选项：打开文件夹和克隆存储库

我们选择打开文件夹，选择自己的工作空间即可

再点击初始化存储库

点击√即可提交

并在旁边的框中输入master并回车

再点击旁边的三个点/拉取，推送/推送

此时，右下角会提示 “存储库未配置任何要推送到的远程存储库”

点击添加远程，再点击上方从Github添加远程

此时会弹出如图所示窗口

此处网站可能被墙，maybe需要挂梯子

选择Allow

点击Continue ,选择程序Vscode

点击打开链接，会返回一个如图的提示，选择打开

此时我们的Vscode就可以获取到github上的库了

这时先不要推代码，放置乱推代码导致代码库被污染

我们来到github建立新仓库

注意：此时可在下面选择Add .gitignore 此选项将会默认配置一些不上传到git的文件，后面也会讲到如何自己添加不想上传的文件

建立好之后回到Vscode选择三个点/拉取，推送/推送到

从Github添加远程，输入刚才创建的库的名称并选择，输入master回车即可

此时便开始推送，若Github上发现没推送，点击vscode左下角这个图标推送

(有可能是一个云朵)

此时回到github即可发现推送完成

添加.gitignore文件

有些文件我们肯定不希望一并被推送上去，比如某些数据等

此时，可以通过添加.gitignore文件实现

直接在工作区根目录下新建文件.gitignore

并以相对路径的方式写入不想上传的文件即可

PS: 可以右键点击不想上传的文件复制相对路径，ctrl+v到gitignore文件中，并在最开始加入一个斜杠”/“，且把后面所有反斜杠”\“全部改为斜杠”/“

添加分支

在vscode左下角点击master 在上方点击添加新分支输入名称回车即可

也可以在github上branch处完成

一般用于不同版本代码管理

拉取他人的代码

先新建立一个工作目录(防止污染)并用vscode打开
